ABOUT PROJECT:
This project is an online news portal.Here I used html,css,python & django.In this website,there is an admin panel and a manager panel.viewers can see the news and send messages to the admin panel. Admin can manage everything. Manager can only add news.


RUN PROJECT:
	1. open the folder in visual studio code.
	2. run python manage.py runserver
	3. go to URL : http://127.0.0.1:8000/



Admin Page URL:
	http://127.0.0.1:8000/panel/
